Structure and Syntax of HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the basic structure and content of the page using a system of "tags." These tags define elements like headings, paragraphs, links, images, and more. This section explains how HTML is structured and the syntax of HTML documents.

HTML Document Structure

An HTML document has a specific structure that defines how web pages should be formatted and displayed in a browser. The basic structure of an HTML document includes several key elements:

1. Declaration

This tag is used to define the document type and version of HTML being used. It is placed at the very beginning of the HTML document. For HTML5, the declaration is:

<!DOCTYPE html>

This tells the browser that the page is written using HTML5, ensuring the page is displayed correctly.

2. <html> Tag

The <html> tag wraps the entire content of the document. It tells the browser that everything inside this tag is HTML content.

<html>
  <!-- HTML content goes here -->
</html>

3. <head> Tag

The <head> section contains metadata and other important information about the web page. This section is not visible to the user but provides necessary details like the title, links to external files (CSS, JavaScript), and meta tags for SEO (Search Engine Optimization).

<head>
  <meta charset="UTF-8">
  <title>My First Web Page</title>
</head>

4. <body> Tag

The <body> section contains all the visible content of the webpage. This is what users see when they visit the page, such as text, images, videos, and links.

<body>
  <h1>Welcome to My Web Page</h1>
  <p>This is a paragraph of text.</p>
</body>

Putting Everything Together

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to My Web Page</h1>
    <p>This is a paragraph of text.</p>
  </body>
</html>

Elements and Tags

In HTML, content is defined using tags, which are special keywords enclosed in angle brackets (<>). Tags come in pairs: an opening tag (<h1>) and a closing tag (</h1>). The opening tag defines the start of an element, and the closing tag defines the end.

Text Elements

<h1> to <h6>: Headings

These tags are used to define headings. <h1> is the highest (most important) level, while <h6> is the lowest level.

<h1>This is a main heading</h1>
<h2>This is a subheading</h2>

<p>: Paragraphs

Defines a paragraph of text.

<p>This is a paragraph of text.</p>

<strong>: Bold Text

Makes text bold and indicates importance.

<strong>This text is important and bold.</strong>

<a>: Links

Used to create hyperlinks to other pages or websites.

<a href="https://www.example.com">Click here to visit Example</a>

<img>: Images

Used to display images on a webpage.

<img src="image.jpg" alt="A description of the image">

<ul> and <ol>: Lists

<ul> defines an unordered list (bulleted list) and <ol> defines an ordered list (numbered list). <li> defines a list item.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Additional HTML Concepts

HTML is not just about adding basic elements. You can also customize the layout and behavior of your content using various attributes and elements:

1. Attributes

Attributes provide additional information about an element. Common attributes include:

  • href: Defines the URL for links.
  • src: Defines the source for images or videos.
  • alt: Provides alternative text for images (useful for accessibility).
  • id: Uniquely identifies an element.
  • class: Allows you to group elements and style them with CSS.

2. Forms

Forms are used to collect user input. You can create forms with <form> and use various input fields such as textboxes, buttons, and radio buttons.

<form action="/submit" method="POST">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username">
  <button type="submit">Submit</button>
</form>

3. Nesting Elements

HTML allows nesting elements inside each other. This is useful for organizing content. For example:

<div>
  <h1>Website Title</h1>
  <p>A description of the website</p>
</div>
